## Goal
Enable reuse and curation of assessment questions over time.

## Scope
- Save items from generated assessments
- Tags: objective, chapter/section, type, difficulty, Bloom, citations
- Search and filter
- Phase 2+: near-duplicate detection and analytics

## Requirements
- Approval state per item (draft/approved)
- Track usage metadata (times used, last used, rating)

## Acceptance Criteria
- Teacher can build a quiz using only approved items.
- Items retain citations and can be re-verified.
